# vcpkg_clean_msbuild

**This function has been deprecated in favor of [`vcpkg_msbuild_install`](vcpkg_msbuild_install.md).**

Clean intermediate files generated by `vcpkg_install_msbuild()`.

## Usage

```cmake
vcpkg_clean_msbuild()
```
