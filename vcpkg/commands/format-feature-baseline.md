# vcpkg format-feature-baseline

## Synopsis

```console
vcpkg format-feature-baseline [<path>]
```

## Description

Formats "ci.feature.baseline.txt" files intended for use with [`x-test-features --ci-feature-baseline`](./test-features.md)
